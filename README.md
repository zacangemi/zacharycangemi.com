# zacharycangemi.com

Source for [zacharycangemi.com](https://zacharycangemi.com) — Zachary Cangemi's personal portfolio.

Static HTML / CSS / JavaScript. No build step. Hosted on GitHub Pages, custom domain via Route 53.

## Structure

- `index.html` — home
- `about.html` — bio
- `skills.html` — skill grid
- `portfolio.html` — featured projects
- `blog.html` — index linking to [blog.zacharycangemi.com](https://blog.zacharycangemi.com)
- `contact.html` — contact form + social links
- `404.html` — custom not-found page
